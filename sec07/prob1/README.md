## メモリのページとは何か
メモリの単位。
以下、p314のコピペ。
OSとCPUが使う単位メモリ。伝統的に4KBか8KBが使われている。
最近のプロセッサは、これらよりも大きなサイズのページを含む複数ページサイズサポートを持つ。
